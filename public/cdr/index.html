Halaman /cdr Siap Dipakai!

Ini adalah folder statis mandiri untuk landing page baru Anda. Anda bisa meletakkan seluruh file HTML, CSS, JS, dan gambar Anda di sini!

Panduan Mengupload File Anda:

  • • Letakkan file index.html Anda di dalam folder /public/cdr/.
  • • Letakkan semua file javascript, css, atau gambar Anda di dalam folder /public/cdr/.
  • • Di dalam file index.html Anda, pastikan pautan (link) untuk asset menggunakan jalur relatif, seperti src="./script.js" atau href="./style.css".
